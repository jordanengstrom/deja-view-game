# DejaView

## Description
This is a flashchard based game where players are shown a series of flashcards/images, and their goal is to put them back in the correct order as quickly and accurately as possible.
Players should aim to get as many points as possible.
The selected difficulty level controls the speed at which the images are shown corresponds to the player, and affects how many points players are given.

## How to Play
[YouTube Tutorial](https://youtube.com/shorts/S1Cl-H31h1w)

1. Select a difficulty
2. Press play
3. Put numbers on the cards in the correct order - the order in which they were shown.

## Difficulty and Scoring Mechanics
Players are scored based on how fast and how accurate they reassemble the order of the cards. This table contains data that should give players an idea of how difficulty affects their score.
| Mode | Speed              | Points Multiplier |
|------|--------------------|-------------------|
| Easy | 60 frames per card | 0.5               |
| Med  | 30 frames per card | 1.0               |
| Hard | 10 frames per card | 2.0               |
